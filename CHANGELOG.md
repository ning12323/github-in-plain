# GitHub in Plain — Changelog

## 1.0.0 — 2026-06-09

Initial release.

- Plain-language GitHub repo explanations for non-programmers (大白话)
- Zero dependencies — Markdown skill only, works in Codex, Claude Code, and Cursor
- Stable 7-section output template with self-validation checklist
- Optional technical 12-dimension report mode
- Examples: FastAPI, Vite, Ollama
